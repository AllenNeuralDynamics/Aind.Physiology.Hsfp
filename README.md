# Getting started

During the early development phase, this repository is intended for internal use only.

## Installing the Bonsai Hamamatsu package

1. Download a nuget package release from github ([e.g.](https://github.com/AllenNeuralDynamics/AllenNeuralDynamics.HamamatsuCamera/actions/runs/21014019222)) that corresponds to the version specified in the `bonsai/Bonsai.config` file.
2. Create a folder named `.tmp` in the root of this repository.
3. Launcher Bonsai, it will attempt to find the Hamamatsu package in the `.tmp` folder.
4. Do NOT commit packages or the `.tmp` folder to git.
